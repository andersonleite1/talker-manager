Talker Manager

Search al talkers

GET - /talker

Ex: http://localhost:3000/talker

Search talker by id

GET - /talker/:id

Ex: http://localhost:3000/talker/5

Search talker by name

GET - /talker/search?q=searchTerm

Ex: http://localhost:3000/talker/search?q=maria

Login

POST - /login

Ex: http://localhost:3000/login

Register talker

POST - /talker

Ex: http://localhost:3000/talker

{ name: "Maria Silva", age: 26, talk: { watchedAt: "22/10/2019", rate: 5 } }

Update talker

PUT - /talker/:id

Ex: http://localhost:3000/talker/5

{ name: "Maria Oliveira", age: 26, talk: { watchedAt: "22/10/2019", rate: 5 } }

Delete talker

DELETE - /talker/:id

Ex: http://localhost:3000/talker/5
